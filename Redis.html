数据库性能调优

Redis的简单认识

数据库

MySQL是关系型数据库，数据存放到硬盘中。读取速度受硬盘转速限制。

关系型数据库是由一张张二维表组成，二维表是Excel表格的。

读取速度都是很快的=》内存

memcached存放数据无法持久化（将数据写到硬盘中）。

Redis数据也是存放到内存中，所以速度快。并且数据可以周期性写到硬盘中，Redis数据可以持久化。

Redis是使用C语言写的。

Redis语法

Redis有五种数据类型，字符串、集合（交集、并集、差集）、有序集、合、list(双向链表)、hash

交集： {a,b,c} n {b,c} = {b,c}

并集： {a,b,c} U {b,c,d} = {a,b,c,d}

差集： {a,b,c} - {b,d} = {a}

每一种数据类型都对应上不同的命令

数据结构

  1. 栈

    特点： 先进后出

  2. 队列(消息队列)

    队列其实就是排队。 特点： 先进先出

  3. 链表

    双向链表/单向链表、循环链表

    单向链表： A => B => C => D

双向链表 ： A <=> B <=> C <=> D

循环链表： A <=> B <=> C <=> D <=> A
